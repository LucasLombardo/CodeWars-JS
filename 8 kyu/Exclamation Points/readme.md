## 8 kyu: Exclamation Points

Link to problem on CodeWars: N/A

### Problem Instructions

Description:
Replace all vowel to exclamation mark in the sentence. aeiouAEIOU is vowel.

Examples

```
replace("Hi!") === "H!!"
replace("!Hi! Hi!") === "!H!! H!!"
replace("aeiou") === "!!!!!"
replace("ABCDE") === "!BCD!"
```
